# GasTag Website

Marketing website for the GasTag iOS app.

## Development

### Prerequisites

- Node.js 20+

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Opens at http://localhost:4321

### Build

```bash
npm run build
```

Static files output to `dist/`.

### Preview production build

```bash
npm run preview
```

## Deployment

Automatically deploys to GitHub Pages on push to `master` via GitHub Actions.
